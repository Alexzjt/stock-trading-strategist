# 筹码分析与主力行为识别 (Chip Distribution & Institutional Behavior)

本规则提炼自筹码分析领域的经典文献。筹码分布反映股票全体投资者在流通盘上的建仓成本和持仓量，其本质是"金钱—筹码的兑换游戏"。通过识别筹码形态变化，可以洞察主力（庄家）的吸筹、洗盘、拉升、出货行为。

---

## 〇、主力的定义与角色

**主力**：在二级市场主动买卖（区别于控股大股东）、持有流通股 **12%~20% 以上**、资金规模以**亿**计的大资金投资者。当前 A 股机构资金可占整体市值 50% 以上，主力是流通股的实际主体。

### 牛市三推动模型

| 推动阶段 | 主体 | 信号含义 |
|:---|:---|:---|
| **第一推动** | 主力机构 | 率先入场，指数脱离底部 |
| **第二推动** | 游资 | 主力拉升后跟进炒作个股 |
| **第三推动** | 公众散户大规模入场 | ⚠️ 主力准备出货的信号 |

> **【导师提醒】** "全民皆股"时你正处于第三推动——这是主力准备离场的前夕。

### 主力分析三要素

1. **进出方向**：吸筹还是出货？
2. **持仓仓位**：重仓还是轻仓？
3. **建仓成本**：主力在哪个价格区间拿筹码？

> **【重要警示】** 主力分析是**牛市工具**。熊市中主力重仓股照样系统性下跌，切勿在熊市滥用。

## 一、筹码形态全分类

### 颜色体系（软件基础）

| 颜色 | 名称 | 含义 |
|:---|:---|:---|
| **大红色** | 密集筹码 | 当前股价上下约 70% 的空间内聚集了 ≥60% 流通筹码，是密集的量化标志 |
| **蓝色** | 死筹 | >60 个交易日未换手，通常是机构/主力底仓 |
| **粉/紫色** | 活筹 | 60 个交易日内换手过，是浮动筹码的主体 |
| **黄色** | 历史对比 | 60 交易日前的筹码状态，用于对比迁移变化 |

分析优先级：**形态与转移方向 > 大红色 / 蓝色占比 > 其他颜色**

### 五种基本形态

**1. 低位密集（形态信号，非直接看涨）**
- **特征**：筹码在相对低位狭窄空间聚集。这仅是筹码在该价位充分换手的形态信号，本身不能直接说明看涨，必须配合向上突破和主力高度持仓验证。
- **审查点**：若用户计划在低位密集区买入，质问：仅低位密集还是已确认向上突破？主力是否高度控盘？

**2. 高位密集（形态信号，非直接看跌）**
- **特征**：筹码在高价区聚集。这仅是高位充分换手的形态信号，不能直接作为看跌判定，必须结合主力是否出货（如出现顺势缓压、线下横盘、下滚峰等出货迹象）或向下破位来做最终研判。
- **审查点**：高位密集区不盲目看空，但若伴随主力出货迹象，则必须严厉警告并否决 ❌。

**3. 低位锁定（强势继续看涨）**
- 股价上涨但低位筹码未上移，主力高度控盘未出货。
- **审查点**：持有此类股票应持有并设移动止损，不要早早获利了结。

**4. 单峰密集（即将出方向）**

> **【核心判断】单峰密集 + 大红色 = 即将出方向（不分涨跌，取决于位置和大盘环境）**

| 单峰位置 | 大概率方向 |
|:---|:---|
| **低位单峰** + 大红色（底部，前期长期下跌） | ↑ 吸筹完成，准备拉升 |
| **高位单峰** + 大红色（高位，已有较大涨幅） | ↓ 出货完成，准备下跌 |
| **中位单峰** + 大红色（中间区域） | ↕ 不确定，需结合大盘判断 |

大盘底部选股：用软件筛选「极值筹码（密集筹码）」，找大红色单峰作为候选池。

**5. 双峰形态（关键博弈区）**
- 上峰为套牢区（压力），下峰为主力成本区（支撑）。双峰之间山谷是真空地带，股价运动阻力极小但触峰即遇抛压。
- **双峰填谷**：股价在双峰之间横盘慢慢填满山谷，是主力高效吸筹的技术标志。

**6. 发散形态**

| 类型 | 出现时机 | 含义 |
|:---|:---|:---|
| 上涨中发散 | 持续上涨过程 | 主力底仓锁低位，筹码向高位健康扩散，属于正常上涨 |
| 下跌中发散 | 持续下跌过程 | 满盘被套，**尚未具备反转条件**，需等底部单峰密集才有信号 |

> **【审查点】** 下跌中发散 ≠ 底部信号。没有底部大红色单峰，不要抄底。

**7. 三峰 / 多峰形态（⚠️ 危险信号）**
- 历史上多个价位均有大批投资者被套，洗盘从未彻底完成，每个峰都是压力区，无干净低位筹码结构。
- **铁律**：三峰及以上 = 洗盘不彻底，反弹逢峰必有沉重抛压。

### 筹码密集的量化定义

> **筹码密集 = 在不超过 20% 的价格空间（对数坐标两格）内，聚集了至少 60%（宽松）至 67%（严格）的流通筹码。**

| 密集类型 | 操作含义 |
|:---|:---|
| **低位密集** | 中性形态。若向上突破且主力高度持仓，方具备翻倍基础（必要非充分条件）。 |
| **高位密集** | 中性形态。若向下破位或伴随主力出货迹象（顺势缓压、线下横盘、下滚峰），则极度危险，坚决不买。 |
| **中价位密集** | 筹码本身无法预判方向，需结合其他维度 ⚠️ |

> **【操作要求】** 分析筹码密集必须使用**对数坐标系**，普通坐标下低价股与高价股的涨跌幅视觉失真，结论不可信。

### 工具适用范围

| 股票类型 | 适用性 |
|:---|:---|
| 有明确主力的中线个股 | ✅ 高度适用 |
| 大盘蓝筹（机构长线） | ✅ 适用，但时间周期拉长（吸筹出货可达 1 年以上） |
| 短线热门个股（高换手率） | ❌ 筹码均摊无密集峰，主力性质无法识别 |
| 长期横盘无明显主力 | ❌ 无主力则无定向操控，随机噪音 |

筹码分析**不适用于美股/港股**（机构主导，散户行为模式不同）。筹码峰的形成准确，但消失不一定准确（各家软件算法不同）。

## 二、识别主力筹码的方法论

### 方法一：行为逻辑法（反证法）

**核心思路**：散户和主力行为逻辑截然不同。当"散户会做但主力不会做"的行为**没有发生**时，逆向推断主力在场。

| 现象 | 散户逻辑 | 反证结论 |
|:---|:---|:---|
| **获利不抛** | 散户获利 10%~30% 急于卖出 | 盈利状态 + 无量上涨或无抛压 → 主力持有 |
| **解套不卖** | 散户解套第一反应"先跑再说" | 股价回到套牢密集区无放量且价格坚挺 → 主力锁定筹码 |
| **逆势飘红** | 大盘大跌时散户恐慌割肉抛售 | 大盘大跌而股价逆势收阳（飘红）或横盘不跌 → 主力资金护盘介入 |

> **【审查点】** "大盘跌它不跌，特别强？"→ 验证：该股是否整体盈利状态？没有抛压且逆势收阳（飘红） = 主力重仓锁定或护盘。

### 方法二：数据解析法（专业工具）

| 方法 | 原理 | 现状 |
|:---|:---|:---|
| 大单还原法 | 碎片化成交记录还原原始大单 | 可用 |
| 交割单分析 | 统计筹码在主力与散户间的流向 | 可用 |
| 基金仓位模型 | 基金重仓股与大盘相关性推算仓位 | 间接可用 |
| 账户统计法 | 直接从交易所获取大账户持仓 | **已不可用**（监管限制） |

> **【审查点】** 用户说"我看主力在买"→ 追问依据：感觉不是证据，行为逻辑推断才可复制。

## 三、主力出货的危险信号

主力高度持仓后，出货有三大核心迹象：
1. **顺势缓压**：大盘上涨时，股价反而逆势下跌（主力趁大盘强劲、市场有接盘侠时默默压低出货）。
2. **线下横盘**：股价跌破无穷成本均线（或重要支撑线）后，在均线下方横盘盘整（弱势派发，无力收回）。
3. **下滚峰**：股价先在高位形成筹码密集峰，随着股价下跌在更低位置形成新的筹码密集峰，原高位密集筹码彻底转移，宣告主力已在低位完成出货撤退。

其他辅助危险信号：
- **高位放量横盘**：有可观盈利 + 高位放量横盘 + 高位密集 = 出货
- **缓慢阴跌（慢熊）**：数月每天跌 1% 左右，属于主力默默出货，而非洗盘。
- **高位散布利好**：基本面利好频出配合高位派发，反而是最危险时刻。
- **天量 = 天价**：高位放出历史级别天量，通常是主力出货完成的标志。

## 四、不同大盘环境下的操作策略

| 大盘阶段 | 操作策略 | 核心逻辑 |
|:---|:---|:---|
| 初期熊市（牛末） | **空仓** | 大家都不信会大跌时最危险 |
| 中期熊市 | 小仓超跌反弹，主要持币 | 资金效率低 |
| 末期熊市 | 寻找逆势不跌的强势股/龙头 | 极度悲观时寻找逆势飘红 |
| 初期牛市 | 做受伤庄股和龙头 | 率先走出底部的是龙头 |
| 中期牛市 | 做低位高控盘庄股，轮动 | 行情火爆买冷门股 |
| 末期牛市 | **分批逃顶** | 反复突破压力线让人麻木时就是顶部 |

### ⚠️ 1% 流动性逃顶极限
在牛市末期（第四推动阶段），散户往往情绪高涨，80%~90% 的账户呈满仓状态。此时市场总市值虽然被推高到天量，但整个市场实际可变现的活跃资金极其有限。当市场风向转变时，所有人都想卖出，但场外已无新增接盘资金。**在顶部能够成功变现逃生的资金往往仅占总市值的 1% 左右**，其余 99% 的资金都将在缺乏流动性的踩踏中被无情套牢。因此，牛市必定以惨烈暴跌结束，且必须跌至低位形成“深坑”，让散户割肉出局后才能开启新轮回。
- **审查点**：在大盘高位或牛市末期，坚决反对满仓操作，必须主动分批锁定利润，保留现金，切忌存有“等我回本就卖”的幻觉。

## 五、成交量（换手率）的深度解读

1. **无量下跌 = 洗盘**：主力未出逃，用少量筹码打压吓走散户
2. **无量上涨 = 高度控盘**：浮动筹码稀少，主力不需要多少资金就能推高
3. **放量可能是对敲骗线**：主力自买自卖制造假象，高位控盘股巨大成交量很多是对倒
4. **无量信号更可靠**：无量无法伪造，比放量更真实

> **【审查点】** 用户说"放量突破"→ 追问：换手率是多少？是真实买盘还是对敲假量？

## 六、散户心理画像与套牢盘规律

**核心规律：散户对套牢的容忍度，与套牢时间成正比，与套牢速度成反比。**

| 散户行为 | 心理原因 | 主力如何利用 |
|:---|:---|:---|
| **浅套 → 一解套就卖** | 煎熬难受，"回本就跑" | 浅套解套位若无放量，说明持有者是主力（解套不卖） |
| **快速深套 → 拒绝割肉，等回本** | 侥幸心理"等反弹就好" | 反弹中散户不卖 → 主力顺利出货（下滚峰机制） |
| **长期深套 → 接受割肉** | 心理防线被时间消磨，"能少亏就走" | 散户主动在低位交出筹码 → 主力低成本建仓（上滚峰机制） |
| **赚 10%~20% → 拿不住** | 害怕"到手的鸭子飞了" | 散户早早离场 → 主力筹码集中 |
| **横盘 → 忍不住卖出** | 无法忍受不确定性 | 主力借此洗盘，清除浮筹 |

> **【导师提醒】** 当用户行为符合上述任一模式（"刚解套赶紧卖"、"赚了一点赶紧跑"），提醒他：你正在按主力希望的方式行动。
>
> 当用户说"等跌下去了我再出"或"反弹一点我就走"，追问：这是有技术支撑的出场位，还是纯粹的回本情结心理锚定？**"等我回本就卖"在熊市或无庄股中，往往要等数年甚至永远等不到。**

## 七、筹码分析的局限性

1. **A 股最有效，美股/港股不适用**（散户行为可预测性不同）
2. **筹码峰形成准确，消失不一定准确**（各家算法的"抛出概率函数"不同）
3. **庄股时代已基本落幕**，量化基金是新时代"庄家"——它们实时分析筹码反向收割散户，散户必须寻找非对称优势（耐心、长期持有）
4. **中位密集是盲区**：筹码分布无法预判中间位置的突破方向，此时必须结合大盘趋势、基本面判断

## 八、导师的筹码审查清单

- 准备买入 → "这只股票筹码是低位密集还是高位密集？有没有配合向上突破或显示出货迹象（顺势缓压、线下横盘、下滚峰等）？"
- 说"主力在吸筹" → "低位无量横盘持续了多久？筹码是否完成低位密集并越过/站稳成本均线？"
- 高位看到"利好消息"想追高 → "高位利好往往是出货掩护，筹码是否出现高位密集与主力出货迹象？"
- 说"放量突破" → "换手率是多少？是真实买盘还是对敲假量？"
- 在熊市频繁操作 → "熊市应该空仓，一年真正出手机会只有几次"
- 操作完全符合散户模式 → "停下来，你正在按主力期望的方式行动——反向思考"

## 九、筹码密集代理指标（Agent 筛选用）

筹码集中度需人工确认（看筹码图 + 股东户数），Agent 通过以下代理指标触发预警：

| 代理指标 | 阈值 | 含义 |
|:---|:---|:---|
| 横盘天数 | ≥ 20 日 | 主力有足够时间吸收浮筹 |
| 横盘振幅 | < 15% | 筹码在狭窄区间形成单峰 |
| 换手率 | 持续 < 3% | 浮筹极少，交投冷清 |
| 量比 | 长期 < 0.7 | 地量 = 没人卖 = 浮筹洗光 |

满足 ≥ 3 条时输出：**⚠️ 疑似筹码集中，请人工确认（单峰密集 + 股东户数持续下降）**

**同板块多选一（Tiebreaker）**：
```
📊 {板块}二选一：
  🥇 {代码A} 涨幅{N}% 量比{x}x 偏离{N}% 横盘{N}日振幅{N}%
  🥈 {代码B} 涨幅{N}% 量比{x}x 偏离{N}% 横盘{N}日振幅{N}%
→ 技术面{A/B}占优，决定性因素是筹码集中度，请人工对比后选择
```

## 十、启动前筹码集中度：涨幅预测器

> **长期窄幅横盘 + 极低换手 + 振幅收窄 → 启动后爆发力远超筹码分散的同类。** 概念龙头筹码分散→放枪后横住；业绩龙头机构锁仓→接力后持续主升。

## 十一、滚峰技术：主力出货 / 建仓的动态识别

> **"滚峰"是筹码峰在时间轴上发生位移的过程，直接揭示主力的出货或建仓行为。** 其本质是大规模换手：旧位置持有者卖出，新位置投资者接盘。

### 1. 下滚峰（看跌信号 — 主力出货定式）

**形态识别（缺一不可）：**
1. 高位形成密集筹码峰
2. 断头铡刀式急速下跌，造成高位套牢盘
3. 快速反弹，但横盘整理，无法创新高
4. 更低的价格区间形成**新的密集峰**，原高位峰完整消失

**逻辑**：被快速套牢的散户期待回本不割肉（回本情结）。急跌→迅速反弹过程中散户几乎不卖，因此换手的是主力，散户被套牢留在了低位。

**操作**：发现标准下滚峰 → 立即止损。下滚峰完成后该股进入无庄熊市，每次反弹触及成本均线即遇压制。

**⭐ 隐蔽出货形态（成本均线判定法）：**

| 步骤 | 表现 |
|:---|:---|
| 第一步 | 高位形成密集筹码峰 |
| 第二步 | 断头铡刀式急速下跌，造成高位套牢盘 |
| 第三步 | 快速反弹但**未越过成本均线**，在成本均线下方横盘出货 |
| 确认 | 筹码峰完整下滚；若股价再度反弹至成本均线附近被压 = 散户**最后逃命窗口** |

> **【审查点】** "急跌后反弹回来了，是不是可以持有？"→ 核查是否发生下滚峰。若确认，反弹至成本均线附近是**最后逃命窗口**，绝非再次建仓信号 ❌。

### 2. 上滚峰（看涨信号 — 主力建仓定式）

**形态识别（三阶段完整路径）：**
1. **成本均线下方横盘**：低位被反复压制，散户被"熊市教育"，形成低位孤立单峰密集（主力低位吸筹完成）
2. **越过成本均线**：股价向上突破，筹码峰开始上滚（长期深套的散户急于获小利卖出，主动交出筹码）
3. **成本均线上方横盘**：形成新的对称密集峰，**吸筹完毕的最终确认**

**⭐ 吸筹完成的三条确认（全满才算）：**
- ✅ 筹码峰在成本均线**上方**形成对称密集（不能在均线下方）
- ✅ 横盘期间成交量极度萎缩（浮筹已被吸尽）
- ✅ 底部孤立单峰依然存在（主力低位建仓筹码未动）

**双重地板结构**：底部孤立单峰 + 均线上方新峰 = 双峰结构，筹码结构转强的技术标志。

> **【审查点】** "涨了一点就卖了"/"横盘忍不住割了" → 警醒：这种行为是否配合了主力完成低位吸筹？能否下次等成本均线上方确认后再决策？

### 3. 真假滚峰辨别

| 判别维度 | 真实滚峰 | 伪滚峰 |
|:---|:---|:---|
| 底部孤立单峰 | **必须有** | 筹码发散或多峰共存 |
| 换手幅度 | 上方峰**彻底消失** | 原有峰仍部分残留 |
| 价格移动幅度 | 完整箱体翻转（15%~30%） | 小幅震荡的筹码平移 |
| 挖坑动作 | 下滚峰必须有（急速下跌制造套牢） | 缓慢均匀的筹码扩散 |

### 4. 吸筹 / 出货时间校准

| 股票类型 | 参考时间 |
|:---|:---|
| 大盘股 / 成交不活跃 | **数月至一年以上** |
| 中盘股 | **数周至数月** |
| 小盘股 / 成交活跃 | **2 周至 1 个月** |

> **【审查点】** 用户说"横盘一个月，吸筹完了"→ 必须追问：大盘股还是小盘股？大盘股一个月在时间维度上可能只完成了吸筹的一小部分。

## 十二、市场训练效应（Market Conditioning）

> **市场会训练投资者形成固定的买卖反应。主力正是利用这种顺向思维实现逆向收割。**

当股票在较长时间内反复呈现某种价格模式（如"每涨 10% 必然回调"），散户会按历史模式提前行动，主力反向操作。

| 散户被训练出的行为 | 逆向推断 |
|:---|:---|
| 前期多次在涨 10%~15% 时卖出 | 本次大量散户出货 → 买入方大概率是主力低位建仓（上滚峰信号） |
| 前期套牢后反弹不卖（等回本） | 本次若形成下滚峰 → 出货方是主力（下滚峰信号） |
| 习惯低位不动（长期深套躺平） | 低位横盘 + 缩量 → 浮筹已清，主力建仓完成 |

> **【审查点】** 用户理由是"这只股票以前每次都这样涨"→ 严肃警告：市场训练效应可能正在对你生效。主力有能力改变历史规律，而散户仍用旧规则交易。

### ⭐ 事不过三原理（求证与踩踏陷阱）
散户的认知模式是“求证型”：一个规律、战法或龙头板块出现第一次时，散户怀疑；出现第二次并被验证赚钱时，散户开始相信；到了**第三次**时，散户彻底完成“求证”，全市场达成共识并疯狂涌入。
然而，一旦某种简单规律或概念被全市场散户“求证”成功，抢筹极其拥挤，主力将彻底失去控盘空间（无法低价拿筹码）。此时主力只有两种选择：**放弃该板块**，或者**顺应一致性预期在高位将筹码彻底倒给散户**，导致第三次尝试成为最惨烈的踩踏陷阱。

## 十三、历史深套盘铁律

> **【铁律】历史深套盘只能被"洗掉"（迫使割肉），不能被"解套"（直接拉回成本价）。**

- 解套盘集中抛售使主力即便托市也难以消化——谁接谁死
- 唯一清除方式：长期震荡让投资者在各价位逐渐割肉
- 完整洗盘需要 **400~600 个交易日（约 2 年）**

**洗盘完成的量化标准（三条全满）：**
- 当前成本位以上套牢盘比例降至 **10% 以下**
- 筹码从多峰发散压缩为**低位单峰密集**
- 成交量极度萎缩（地量），浮筹接近枯竭

**实用口诀**：看套牢盘在哪里，就知道这波反弹最高能到哪里。

> **【审查点】** 用户说"跌了很多了，应该要反弹了"→ 追问：上方还有多少历史套牢盘？未清干净之前，每次反弹都可能是套牢盘逃命机会，而非新牛市起点。

## 十四、无庄股与主力吸筹“挖坑定式”

在熊市或长期盘整中，识别股票是否有主力控盘、是“无庄股”还是“庄股吸筹完成”，需结合 **无穷成本均线（CYC）** 与“大坑”形态判定。

### 1. 无庄股特征（绝对回避 ❌）
- **现象**：股价长期被**无穷成本均线（CYC）**死死压制。每次反弹到无穷成本均线附近即遇抛压回落。
- **本质**：没有主力在场（散户重仓股），或者是主力完全放弃的死股。散户多空换手毫无规律，股价随大盘随机波动，稍有反弹即遭解套盘踩踏。
- **审查点**：如果用户想买入这种被无穷成本均线长期压制、且无异动信号的个股，予以**坚决否决**。

### 2. 主力吸筹“挖坑定式”（经典建仓信号 💰）
这是主力利用散户绝望心理进行“极限洗盘与高效拿筹码”的标准模式：

| 步骤 | 市场表现 | 主力与散户心理博弈 |
|:---|:---|:---|
| **第 1 步：长期打压** | 股价长期受制于无穷成本均线，散户被磨平耐心，主力低位零星吸纳。 | 散户被熊市教育，习惯了“高抛低戏”的波段操作。 |
| **第 2 步：顺势砸坑** | 借助市场系统性暴跌或利空，股价急速砸出一个**大坑（跌幅通常 >20%）**。 | 砸出恐慌盘，逼迫坚守的散户在深套中彻底绝望并割肉。 |
| **第 3 步：快速回收** | 股价在极短时间内拉回并**越过无穷成本均线**。 | 主力通吃恐慌割肉筹码，价格快速脱离低位大坑。 |
| **第 4 步：横盘锁定** | 股价在无穷成本均线**上方**或坑沿上眼进行窄幅横盘整理（通常 10-15 个交易日）。 | 散户解套盘和短线获利盘抛售，主力在均线上方强势接盘。 |

**吸筹完成的确认指标：**
- **横盘缩量**：横盘期间成交量极度萎缩（地量），说明浮动筹码已基本被主力吸尽，散户不再卖，主力也锁仓。
- **筹码集中**：底部低位单峰密集，且在均线上方形成对称的新密集峰。
- **操作建议**：一旦确认“均线上方强势横盘 + 地量锁仓”，即为主力吸筹完毕准备拉升的临界点。

> **【审查点】** 用户说“之前跌了很多，最近反弹快，要走牛了？” $\rightarrow$ 追问：反弹是否越过了无穷成本均线？在均线上方是否经历了**缩量横盘整理（10日以上）**？如果没有，仅是无庄股的超跌反弹或“挖坑失败”（反弹遇阻回落），切勿追高。

## 十五、筹码分布的三大核心用途

**用途一：知己知彼**
直接显示全市场持仓成本分布——谁被套、谁在盈利、谁会在哪个价位卖。这是裸 K 线图无法提供的维度。

**用途二：否定可能性**
- 上方大量套牢盘 → **否定**短期能突破该价位
- 低位已形成密集单峰 → **否定**无底洞式下跌
- 多峰发散 → **否定**近期有主力护盘

> **筹码分析能否定，但无法肯定——告诉你哪里不能去，不能告诉你哪里一定能去。**

**用途三：判断主力控盘度**
- 单峰密集（低位锁定） = 主力在场，底仓稳固
- 多峰发散 = 主力不在场，无庄熊市
- 主力在场 ≠ 股价一定上涨；**主力不在场 = 股价难以持续上涨**

> **【审查点】** 用筹码分布时，时刻问自己："我现在是在否定一种可能性，还是在肯定一种必然性？"前者正确，后者是过度解读。

## 十六、牛熊转化的筹码状态四阶段

> **识别当前处于哪个阶段，是判断大方向的核心工具。以周为单位观察筹码迁移方向，不能从单一交易日判断。**

| 阶段 | 筹码特征 | 判断信号 |
|:---|:---|:---|
| **① 吸筹期（熊末牛初）** | 上方多峰发散 → 套牢者割肉 → 底部逐渐形成低位大红单峰 → 蓝色死筹升高 | 上方峰持续消失 + 底部红色单峰成型 |
| **② 拉升期（初期牛市）** | 底部单峰形成后筹码上移（红变蓝/粉）；低位主力底仓始终未动（获利不抛） | 蓝色死筹稳定 + 筹码向上健康发散 |
| **③ 牛市调整（中期）** | 正常：局部向下转移→大红→再向上突破消退；危险：底部未洗净强行冲顶，出现三峰多峰 | 出现"上大峰/下小峰"不对称结构 = 进入调整，不加仓 |
| **④ 下跌期（熊市）** | 高位大红出货 → 快速被套（暴跌：情绪性，可能是阶段底；缓跌：主力出货，不接刀） | 缓慢阴跌 + 筹码扩散无密集峰 = 漫长熊市信号 |
