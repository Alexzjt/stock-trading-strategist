# 个股信号分类框架 (Signal Classification Framework)

> **用途**：定时扫盘任务（cron job）或批量分析时，对每只股票输出明确的 💰入场 / 🟡持有 / ❌卖出 三选一信号。
> **核心原则**：信号质量决定"做不做"，价格偏离决定"做多少"。
> **知识来源**：日本蜡烛图技术新解（尼森）+ 股票操作学（张龄松/罗俊）+ 炒股的智慧

---

## 信号判定优先级

```
优先级：❌卖出 > 💰入场 > 🟡持有
```

先检查卖出，再检查入场，都不满足默认持有。

---

## ❌ 卖出信号 (Sell Signal)

满足**任意一条**即触发：

### S1 — 趋势转空
短期趋势从 Uptrend/Consolidation 变为 **Downtrend**。

### S2 — 跌破生命线
收盘跌破 MA10，且**次日未收回**（需要连续2天确认，单日假跌破不触发）。

> ⚠️ **数据门槛**：MA10 必须已有 ≥ 10 个交易日的数据积累。上市初期/数据不足时 S2 自动静默，避免基于不完整均线的误报。

### S3 — 高位 K 线反转（共7种形态）

以下任一形态出现在**上涨高位**（偏离MA10 > 5%，或近5日累计涨幅 > 10%）：

| 形态 | 来源 | 判定标准 |
|:---|:---|:---|
| **流星线/长上影线** | 蜡烛图 + 股票操作学 | 上影线 ≥ 实体 2x；高位长上影线 = "多头被空头打回来，主力撤退" |
| **吊颈线** | 蜡烛图 | 高位长下影线小实体；**需等次日阴线确认**后触发（"专门骗接盘的极度危险信号"） |
| **黄昏星** | 蜡烛图 | 大阳线 + 跳空星 + 大阴线，"K线中最坚实的顶部反转" |
| **看跌吞没** | 蜡烛图 | 阴线实体完全盖住前一根阳线 |
| **乌云盖顶** | 蜡烛图 | 阴线扎入前阳线实体一半以上 |
| **三只乌鸦** | 蜡烛图 | 连续三根阴线，强势看跌 |
| **高位平头顶** | 蜡烛图 | 连续两根K线高点几乎相同，出现在重要阻力位 |

> ⚠️ 单个包孕线(Harami) ≠ S3。包孕线只代表动力衰竭，次日低开低走才升级为卖出。

### S4 — 天量冲高回落
成交量 > 5日均量 **2.5x**，且收盘 < 开盘 / 上影线 > 实体 2x。
（"高位放量滞涨/冲高回落 = 主力出货" —— 股票操作学）

> ⚠️ **天量以绝对换手率为准**：量比>2.5x 但换手率 < 8% 时，因基量低导致倍率虚高，不构成 S4。当日换手率 > 10% 才判定为真天量。
> 🚫 **ETF 不适用 S4**——ETF 换手率天然高于个股，天量规则对 ETF 无意义。

### S5 — 止损触发
收盘价跌破预设止损位。

### S6 — 乖离过大（葛南维卖出法则1）
偏离 MA10 **> 15%**，或连续5日累计涨幅 > 20%。
（"股价远离均线，近期买入者获利极丰，随时有巨大获利回吐卖压" —— 股票操作学）

### S7 — 形态破位（颈线跌破）
收盘价跌破**关键颈线**：M头 / 头肩顶 / 三重顶的颈线。
（"只要跌破颈线，无论亏损多少，立刻无条件清仓！" —— 股票操作学）

### S8 — 岛形反转 / 竭尽缺口
高位出现跳空缺口后，几天内不仅回补缺口，且反向跳空下跌。
（"最致命的暴跌前兆，必须清仓逃命" —— 股票操作学）

### S9 — 量价背离与均线受阻（葛南维卖出法则3+4 + 股票操作学）

满足任一即触发：

| 子条件 | 判定 | 来源 |
|:---|:---|:---|
| **无量空涨** | 股价创新高但量比5日 < 0.7（"买气已竭，随时见顶"） | 股票操作学 |
| **反弹遇阻** | Downtrend中，股价反弹触及MA10/MA50后回落（"均线向下时的反弹全是逃命点"） | 葛南维卖3 |
| **假突破** | MA仍处于下跌趋势，股价短暂突破MA10后快速回落（"均线向下时的突破全是诱多"） | 葛南维卖4 |

## 💰 入场信号 (Entry Signal)

### 基础入场条件（E1–E4，需同时满足）

| 编号 | 条件 | 说明 |
|:---|:---|:---|
| E1 | 趋势为 Uptrend | MA10 方向向上（当前 MA10 > 5日前 MA10）+ 近5日高点/低点抬高 + 收盘站上 MA10。**禁止用 MA5 短期波动替代**——Downtrend 中的 1-2 日反弹不构成 Uptrend |
| E2 | 收盘 > MA10 或 回踩 MA10 站稳 | 在均线上方，或刚回踩未跌破 |
| E3 | 无卖出信号 | 当日不触发 S1–S9 任一条件 |
| E4 | 成交量正常 | 量比5日 0.3–2.5，非极度缩量也非天量。<br>⚠️ 低换手率小盘股（日常换手<3%）因基量低，量比容易虚高。量比>2.5x但绝对换手率<8%时，降级为 🟡谨慎入场而非否决。天量以绝对换手率为准（>10%警戒，>15%危险）。<br>**ETF 不适用天量规则**——ETF 换手率天然高于个股。 |

### 入场分级（按偏离 MA10）

| 偏离 | 分级 | 操作 |
|:---|:---|:---|
| ≤ 8% | 💰 **标准入场** | 标准仓位，止损 MA10 下方 2% |
| 8%–12% | 🟡 **谨慎入场** | 半仓试探，止损放宽到 MA50 |
| > 12% | ⏸️ **等回调** | 信号好但太贵，标记候选等回踩 |

### 加分项（不改变信号结论，但增强信心）

- MA50 金叉 MA200（中期趋势确认）
- 1月均量较3月均量放大 1.5x+ 且价格同步上行
- 属于用户龙头列表

### 底部 K 线确认信号（满足任一条 + E1–E4 → 入场信心大幅增强）

以下形态出现在**下跌末期或低位**（偏离MA10 < -5%，或前期有明显下跌）：

| 形态 | 来源 | 判定标准 |
|:---|:---|:---|
| **启明星** | 蜡烛图 | 大阴线 + 跳空星 + 大阳线，"最坚实的底部反转" |
| **看涨吞没** | 蜡烛图 | 阳线实体完全盖住前一根阴线 |
| **锤子线/长下影线** | 蜡烛图 + 股票操作学 | 下影线 ≥ 实体 2x；"探底反击，多头吹响反攻号角" |
| **刺透形态** | 蜡烛图 | 阳线扎入前阴线实体一半以上 |
| **低位平头底** | 蜡烛图 | 连续两根K线低点几乎相同，出现在支撑位 |

### 葛南维特殊买入法则

| 法则 | 条件 | 操作 |
|:---|:---|:---|
| **假跌破买入**（买3） | MA仍处于上升趋势，股价短暂跌破MA10后收回 | 🟡 谨慎入场，这是洗盘不是真破 |
| **极度乖离买入**（买4） | 偏离MA50 < -20%，暴跌后极度超卖 | ⚠️ 仅限抢反弹，轻仓+严格止损 |

### ⭐ E5 — 缩量回调结束入场 (Pullback End Entry)

这是经多重经典理论交叉验证的高胜率入场信号。核心逻辑：

> **量缩买点（A股实战）** ∩ **十字星刹车（蜡烛图）** ∩ **支撑线企稳（炒股智慧）** = 回调结束

**理论依据**：
- `wisdom/02_technical_rules.md`："正常升势：上涨（量大）→ 回调（量缩到启动前水平）→ 再次上涨（量大）"
- `stock_trading_practice_tw.md`："先见量，后见价" + "多头市场回调时成交量必须迅速萎缩"
- `japanese_candlestick_techniques.md`：十字星/小实体 = "趋势的刹车"，原有动力衰竭

**触发机制**（两阶段，适配定时扫盘无法预测次日）：

#### 第一阶段：候补识别（当日 → 🟡持有⚠️回调尾声）

当日满足以下**全部四项**时，标记为候补：

| 条件 | 判定标准 | 数据来源 |
|:---|:---|:---|
| **趋势前提** | 前期有上升趋势（近20日有明确的高点抬高/低点抬高，或区间涨幅 > 5%） | analyze_trend.py 趋势判断 |
| **位置到位** | 当前价回调至关键支撑位附近（MA20 ± 3%、MA50 ± 3%、前波谷 ± 2%、或上升趋势线 ± 2%） | MA值 + 关键位列表 |
| **量能枯竭** | 当日量缩至近20日最低量附近（量比20日 < 0.7，或量比5日 < 0.5）——本质是缩回上一波上涨启动前的地量水平 | analyze_trend.py vol_ratio 字段 |
| **企稳K线** | 收小阳线（实体 > 0 且 ≤ 前日振幅的 1/3）或 十字星（实体 ≤ 上下影线较小者，且实体/前日实体 < 0.5） | K线形态判断 |

**当日输出格式**：
```
🟡持有⚠️回调尾声 {代码} {名称} {收盘价} - 地量{倍数}x 缩至{支撑位类型}({价位}) {K线形态} → 等明日放量阳线确认
```

> ⚠️ 十字星出现在"真空地带"（不在任何关键支撑位附近）→ 不触发，只是普通噪音。

#### 第二阶段：确认入场（次日 → 💰入场）

前一日有 [回调尾声候补] 标记，且**今日满足以下全部条件**时，升级为入场：

| 条件 | 判定标准 |
|:---|:---|
| **放量** | 今日量比5日 > 1.2，或量明显大于昨日 |
| **阳线确认** | 今日收盘 > 开盘，且收盘 > 昨日收盘 |
| **无卖出信号** | 不触发 S1–S8 任一条件 |
| **位置未恶化** | 未跌破昨日的关键支撑位（收盘 > 支撑位 - 1%） |

**入场分级**：

| 偏离 MA10 | 分级 | 操作 |
|:---|:---|:---|
| ≤ 8% | 💰 **回调结束入场** | 标准仓位，止损设昨日最低点下方 2% |
| 8%–12% | 🟡 **谨慎入场** | 半仓试，等再回踩 |
| > 12% | ⏸️ **等回踩** | 标记追踪 |

**输出格式**：
```
💰回调结束入场 {代码} {名称} {收盘价} - 昨日地量{倍数}x缩量{支撑位}企稳 今日放量阳线确认 偏离{偏离%}% → {分级提示}
```

#### 候补失效条件

出现以下任一情况，取消候补标记：
- 次日收阴线且收盘 < 支撑位（破位）
- 连续3日未出现确认阳线（时间失效，可能继续阴跌）
- 触发任何卖出信号（S1–S8）

#### 止跌变体：两种"止跌"，同一逻辑

用户发现 E5 的本质不是"缩量"，而是**"止跌 + 阳线确认"**。止跌的 K 线可以有两种长相，共享完全相同的第二阶段确认：

| 变体 | 止跌日特征 | 确认日 | 适合标的 | 案例 |
|:---|:---|:---|:---|:---|
| **A. 缩量企稳**（经典 E5） | 缩量至地量 + 小阳/十字星，卖方枯竭 | 次日放量阳线确认 | 蓝筹/白马/慢牛趋势股 | — |
| **B. 恐慌反包**（用户杀手锏） | 放量恐慌阴线（量比>1.5x，跌>3%），恐慌盘被杀光 | **次日放量大阳反包**（量比>1.5x，收盘完全吞没恐慌阴线）= 当日即确认 | 游资票/题材票/中小盘 | 金安国纪 002636：5/21放量跌-4.6%→5/22放量反包+8.4%→一周+16% |

> 两个变体共享同一底层逻辑：**回调到支撑位 → 出现止跌信号 → 阳线确认 = 入场**。区别只是止跌那根K线是"没人卖了"（缩量）还是"有人全收了"（恐慌+反包）。

**恐慌反包入场规则**：反包日收盘确认，偏离 ≤10% = 标准仓位，止损设恐慌日最低点 - 2%。

**恐慌反包输出格式**：
```
💰回调结束入场(恐慌反包) {代码} {名称} {收盘价} - {恐慌日日期}放量{x}x跌至{低点} {反包日日期}放量{x}x反包 偏离{偏离%}%
```

### ⭐ E6 — 回抽测试买点 (Throwback Entry)

**来源**：`classic_trend_analysis.md §回抽测试`

股价突破关键阻力线（颈线/前高/形态上轨）后，会出现一个缩量回踩动作，重新测试原阻力线（现已变为支撑线）。**只要回踩不破**，再次启动时就是绝佳的安全买点。

**触发条件**（需全部满足）：

| 条件 | 判定标准 |
|:---|:---|
| **有效突破在先** | 此前已发生有效突破（收盘站稳原阻力线上方 3% + 至少2天），突破时放量 |
| **回踩缩量** | 当前回踩过程中成交量明显萎缩（量比5日 < 0.8） |
| **回踩不破** | 最低价未跌破原阻力线（现支撑线），或盘中跌破但收盘收回（"腿"在下方） |
| **再次启动** | 今日收阳线（收盘 > 开盘），且收盘站稳支撑线上方 |

**入场分级**：同 E1–E4 通用分级（偏离 MA10 ≤8% = 💰标准，8-12% = 🟡谨慎，>12% = ⏸️等回踩）

**输出格式**：
```
💰回抽确认入场 {代码} {名称} {收盘价} - 突破后缩量回踩{支撑位}不破 今日阳线再启动 偏离{偏离%}% → {分级提示}
```

**与 E5 的区别**：E5 是上升浪中的自然回调 → 缩量企稳 → 再涨；E6 是**先有突破事件** → 突破后回抽 → 再涨。E6 的信号强度高于 E5，因为多了一层"突破已被验证"的确认。

### ⭐ E7 — 诱空陷阱买入 (Bear Trap Entry)

**来源**：`wisdom/02_technical_rules.md §危险信号识别`

这是大户制造恐慌低价吸筹的经典手法。股价**跌破支撑线后交易量大增，但股价随即弹回支撑线之上**——这是恐慌抛盘被主力全部吃掉的信号。

> 原文："10次试了9次赚"

**触发条件**（需全部满足）：

| 条件 | 判定标准 |
|:---|:---|
| **假跌破** | 盘中或收盘跌破关键支撑位（MA20/MA50/前波谷/形态颈线），但**当日或次日收盘收回支撑线上方** |
| **放量恐慌** | 跌破时成交量显著放大（量比5日 > 1.5，或当日量为近10日最大） |
| **快速收回** | 从跌破到收回不超过2个交易日 |
| **前期趋势健康** | 跌破前处于 Uptrend 或 Consolidation，非长期 Downtrend |

> ⚠️ **关键区别**：诱空的前提是"跌破后弹回"。如果跌破了、放量了、但没弹回来 → 那是真破位（S2/S7），不是诱空。

**入场分级**：
| 偏离 MA10 | 分级 | 操作 |
|:---|:---|:---|
| ≤ 5% | 💰 **诱空入场** | 标准仓位，止损设本次假跌破最低点 - 2% |
| 5%–10% | 🟡 **谨慎入场** | 半仓 |
| > 10% | ⏸️ **等回踩** | |

**输出格式**：
```
💰诱空入场 {代码} {名称} {收盘价} - 假跌破{支撑位}放量弹回收复 偏离{偏离%}% → {分级提示}
```

**风险提示**：诱空属于高风险高回报信号。如果弹回后次日又跌回支撑线下方 → 止损离场，不要犹豫。

### ⭐ E8 — 中继形态突破入场 (Continuation Pattern Breakout)

**来源**：`classic_trend_analysis.md §对称三角形` + `stock_trading_practice_tw.md §整理形态`

上升趋势中，股价经过旗形/三角形/矩形整理后，放量突破形态上轨 → 是趋势继续的信号。

**触发条件**（需全部满足）：

| 条件 | 判定标准 |
|:---|:---|
| **形态识别** | 前期有可辨识的整理形态：上升旗形（高点低点同步下移但夹角小）、对称三角形（高点降低+低点抬高收窄）、矩形箱体（水平上下轨） |
| **突破位置合理** | 突破发生在形态 1/2 到 3/4 处（"价格走到三角形尖端才突破 → 无力，无效"） |
| **放量突破** | 突破上轨日成交量显著放大（量比5日 > 1.3，或较形态构筑期均值放大 > 50%） |
| **收盘确认** | 收盘价站稳上轨上方（盘中突破但收盘跌回 → 不触发） |

**入场分级**：同 E1–E4 通用分级

**输出格式**：
```
💰形态突破入场 {代码} {名称} {收盘价} - {形态类型}放量突破上轨{价位} 偏离{偏离%}% → {分级提示}
```

**与 E1–E4 的关系**：E8 是 E1–E4 的强化版——形态突破时通常也满足 E1（Uptrend），但 E8 额外要求形态识别+突破位置合理+放量，过滤掉假突破，信号质量更高。

## 🟡 持有不动 (Hold)

不触发卖出，也不满足入场 = 默认持有。

| 场景 | 处理 |
|:---|:---|
| Uptrend + 无反转信号 + 未触及止损 | 🟡 持有，不需要操作 |
| 包孕线/十字星在上升趋势中 | 🟡 持有⚠️关注，次日确认方向 |
| 缩量回调但未破 MA10 | 🟡 持有，正常调整 |
| 缩量至地量 + 小阳/十字星 + 关键支撑位（E5候补） | 🟡 持有⚠️回调尾声候补 → 等明日放量阳线确认升级为 💰回调结束入场 |
