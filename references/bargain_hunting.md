# 捡漏框架：财报噪音 vs 真暴雷

## 核心逻辑

市场经常对财报做出过度反应。当一只股票因为季报/年报"亏损"而暴跌时，高手的第一反应不是恐慌，而是**区分这亏损是会计噪音还是经营恶化**。

**一句话判别：**

> 净利润亏损 + 经营现金流大幅为正 = 会计噪音（折旧摊销等非现金支出）  
> 净利润亏损 + 经营现金流也为负 = 真暴雷

## 三步验证法

### 第1步：现金流验证（最核心）

对比净利润和经营现金流：

| 净利润 | 经营现金流 | 判断 | 行动 |
|--------|-----------|------|------|
| 亏损 | 大幅为正（>净利润绝对值） | ✅ 会计噪音（折旧/摊销/减值计提） | 进入第2步 |
| 亏损 | 也为负或接近零 | ❌ 真亏损，经营出问题 | 一票否决 |
| 亏损 | 为正但不大 | ⚠️ 需进一步看营收和毛利 | 进入第2步谨慎判断 |

**原理**：折旧摊销是非现金支出，会计上扣利润但不影响现金流。重资产行业（制造业、玻璃/面板/芯片）折旧体量极大，Q1集中计提就能把账面利润打成负，但实际经营毫发无伤。

案例：蓝思科技300433 Q1 2026 — 净利润-1.5亿，经营现金流+20.14亿，差额≈21.6亿 = 折旧摊销。

### 第2步：营收验证

看营收是否稳定或增长：

- 营收持平或同比增长 → ✅ 市场份额没丢，客户没跑
- 营收暴跌（>20%同比↓） → ❌ 即使现金流没问题，需求端出问题了

### 第3步：毛利率验证

看毛利率是否稳定：

- 毛利率持平或小幅波动 → ✅ 产品竞争力没变
- 毛利率崩塌（>5个点暴跌） → ❌ 可能是降价保量，竞争恶化

## 操作流程

```
季报亏损 → 市场暴跌
    │
    ├─ 经营现金流 > 0 且 营收稳定 且 毛利率稳定
    │   └─ ✅ 会计噪音 → 加入观察清单
    │       └─ 等待技术面底部确认（双底/头肩底/放量突破MA10）
    │           └─ 择机买入，止损设底部下方-3%
    │
    └─ 经营现金流 ≤ 0 或 营收暴跌 或 毛利率崩塌
        └─ ❌ 真暴雷 → 远离，不抄底
```

## 风险提示

- 会计噪音不等于股价一定会涨回来。市场情绪可能持续压制。
- 必须等**技术面底部确认**再入场，不要在暴跌途中接飞刀。
- 商誉过高的公司（>净资产30%），减值风险会反复出现，慎用此框架。
- 本框架适用于重资产制造业。轻资产/服务类公司折旧占比小，亏损就是真亏损。
